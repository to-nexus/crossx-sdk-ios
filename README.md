# CROSSx iOS SDK

The CROSSx Swift SDK is a Swift library client for CROSSx that allows you to add secure authentication, non-custodial embedded wallets, and powerful user management into your iOS application.

## Installation

### Swift Package Manager

Add the following to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/to-nexus/crossx-sdk-ios.git", from: "1.0.0")
]
```

Or add it directly in Xcode: File > Add Package Dependencies

### CocoaPods

Add to your `Podfile`:

```ruby
pod 'CROSSxSDK', '~> 1.0.0'
```

## Usage

See CROSSx's iOS SDK guide for setup and usage.
